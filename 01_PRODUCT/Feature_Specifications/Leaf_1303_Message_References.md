## 1. Purpose
Agents cite sources for every claim. Citations are inline, numbered, and clickable.

## 2. Functional requirements
- Numbered references `[1]`, `[2]`, etc.
- Hover tooltip with title + URL.
- Click opens source.
- Sources listed at end of message.

**Acceptance:**
- [ ] Citations render inline.
- [ ] Sources always listed for research-grade answers.

## 3. Out of scope
- Inline image citations (H2).

## 4. Acceptance criteria summary
- [ ] All factual claims cited.
