## Maintenance rules

1. New doc gets next free number in its range, **first** add the row to the registry, **then** create the file.
2. Cross-reference IDs (e.g., "Depends on NX-ARCH-0101") must be present in the doc's identity table.
3. Status changes (Draft → In Review → Complete) update both the registry and the doc's own header.
4. Deprecated docs stay in the registry with status 🔴 Deprecated and the new path if relocated.
